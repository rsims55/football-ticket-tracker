# Project Overview

This repository tracks college football ticket prices, trains prediction models, and produces weekly reports.

## Quick start (Linux)
1) Run setup: `bin/setup_linux.sh`
2) Add secrets to `.env`
3) Run all: `bin/run_all_linux.sh`
4) Launch GUI: `bin/run_gui_linux.sh`

## Quick start (Windows)
1) Run setup: `bin\setup_windows.cmd`
2) Add secrets to `.env`
3) Run all: `bin\run_all_windows.cmd`
4) Launch GUI: `bin\run_gui_windows.cmd`

## Key paths
- `src/` — application code
- `data/` — snapshots, schedules, and model data
- `reports/` — weekly reports
- `assets/` — icons and UI assets
- `scripts/` — maintenance utilities
